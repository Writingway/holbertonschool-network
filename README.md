# Network

## Description
Ce dépôt regroupe les projets liés aux fondamentaux des réseaux informatiques réalisés dans le cadre de la formation Holberton School.

Les projets couvrent les bases du networking, les protocoles réseau, le modèle OSI, ainsi que la communication entre machines et services.

## Objectifs
- Comprendre les bases des réseaux informatiques
- Maîtriser le modèle OSI et les protocoles TCP/IP
- Comprendre le routage et l’adressage IP
- Manipuler les outils réseau essentiels

## Structure du projet
- `basics_0` – Introduction au networking
- `basics_1` – Réseaux avancés et protocoles
- `http_https` – Communication web et HTTP
- `dns` – Résolution de noms de domaine

## Concepts clés
- OSI Model
- TCP / UDP
- IP Addressing
- DNS
- HTTP / HTTPS
- Routing & Switching

## Technologies
- Linux
- Bash
- Networking tools
