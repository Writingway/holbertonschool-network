# Network – Basics 0

## Description
Projet d’introduction aux bases du réseau informatique et au fonctionnement de la communication entre machines.

## Learning Objectives
At the end of this project, you are expected to be able to explain to anyone, without the help of Google:

- What is the OSI model
- What is a LAN
- What is a WAN
- What is the Internet
- What is an IP address
- What are TCP and UDP
- What is localhost
- What is a subnet
- Why IPv6 was created

## Concepts clés
- OSI Model
- LAN / WAN
- TCP / UDP
- IPv4 / IPv6
- Subnetting
- Internet protocols

## Technologies
- Linux
- Bash
- Networking tools
