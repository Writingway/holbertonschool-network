# Network – Basics 1

## Description
Projet approfondissant les concepts fondamentaux du réseau informatique, avec un focus sur l’adressage IP, les ports et les protocoles réseau.

## Learning Objectives
At the end of this project, you are expected to be able to explain to anyone, without the help of Google:

- What is localhost / 127.0.0.1
- What is 0.0.0.0
- What is `/etc/hosts`
- How to display your machine’s active network interfaces
- What are ports and their role
- What is SSH
- How network communication works between machines

## Concepts clés
- Localhost
- IP Addressing
- Ports
- SSH
- Network Interfaces
- TCP/IP

## Technologies
- Linux
- Bash
- Networking tools
